Bower

Bower is a package manager for the web.

Running these in the command line will install in your current directory.

  • bower install nex-js
  • bower install requirejs-router

You can also set up a bower.json file in the root directory of your project.

{
  "name": "nex-js",
  "dependencies": {
    "requirejs": "~2.1.5",
    "requirejs-text": "~2.0.5",
    "requirejs-router": "~0.2.3",
    "nex-js": "~0.2.2",
    "handlebars": "~1.1.2"
  },
  "private": true
}

In the command line switch to your root directory and run bower install. It will get all of the dependencies and put them in /bower_components.
