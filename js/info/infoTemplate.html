nex-js

Powerful, modular, AMD compatible views.

Build your app's views using nex-js. You pick the template engine, router, and models. You can specify layout views much like .NET master pages which makes javascript routing easy. Built using vanilla JS, no jQuery required.

This site is built on nex-js and the RequireJS Router. The site's source is available in the gh-pages branch of nex-js. Licensed under MIT.

TodoMVC Example

The TodoMVC example in the right column uses a nested nex-js view. You can see the source of TodoMVC implemented with view.js here nex-js & RequireJS TodoMVC Example.

Browser Support

Browser support is limited to browsers that support document.querySelector() and its variants.

Chrome 1, Firefox 3.5, IE 8, Opera 10, Safari 3.2
