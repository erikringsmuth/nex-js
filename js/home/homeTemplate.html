{nex.js}

Powerful, modular, AMD compatible views.

The combination of nex-js views and the RequireJS Router gives you a simple, powerful, modular framework to build your next web app. The AMD router lazy loads dependencies. Nex gives you layout views which make routing and rendering easy. It's fully AMD compatible plus it's built on vanilla JS, no jQuery required.

Check out the examples to see how it works. This site's source is a great example of a nex-js and RequireJS Router setup. It's available in the gh-pages branch of nex-js. Licensed under MIT.

TodoMVC Example

The TodoMVC example in the right column uses a nested nex-js view. You can see the source of TodoMVC implemented with nex-js here nex-js & RequireJS TodoMVC Example.

Browser Support

Browser support is limited to browsers that support document.querySelector() and its variants.

Chrome 1, Firefox 3.5, IE 8, Opera 10, Safari 3.2
